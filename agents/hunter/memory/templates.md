# Templates de messages Hunter

## Restaurant

Template de base:
```
Bonjour,

Je suis tombé sur {nom} à {ville} en cherchant où manger dans le coin.

Je me suis rendu compte que vous n'aviez pas encore de site web — pourtant ça pourrait vraiment aider les clients à vous trouver et voir votre carte.

Je crée des sites simples et rapides pour les restaurants, livrés en quelques jours.

Si ça vous intéresse, je peux vous montrer un exemple adapté à un restaurant comme le vôtre 👍
```

## Artisan

Template de base:
```
Bonjour,

En cherchant des {type} à {ville}, je suis tombé sur {nom}.

J'ai remarqué que vous n'aviez pas de site web — pourtant beaucoup de clients cherchent des artisans urgents en ligne.

Je crée des sites simples et professionnels pour les artisans, livrés rapidement sans prise de tête.

Ça vous dirait de voir ce que ça pourrait donner pour votre activité ?
```

## Coaching / Bien-être

Template de base:
```
Bonjour,

Je découvre votre activité à {ville} — {nom} a l'air super intéressant.

Je me disais qu'un petit site simple pourrait vraiment aider à montrer votre expertise et attirer de nouveaux clients.

Je crée des sites vitrines rapides pour les pros du bien-être.

Si ça vous tente, je peux vous montrer un exemple adapté à votre univers 👍
```
