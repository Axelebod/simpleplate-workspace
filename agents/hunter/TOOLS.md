# TOOLS.md - Hunter

## Sources d'inspiration

### Templates par secteur

**Restaurant:**
- Angle: "Clients qui cherchent où manger à {ville}"
- Bénéfice: Visibilité sur Google Maps, avis clients
- CTA: "Montrer un exemple de site resto"

**Artisan (plombier, électricien...):**
- Angle: "Clients qui cherchent un artisan urgent"
- Bénéfice: Crédibilité, disponibilité visible
- CTA: "Exemple adapté aux artisans"

**Coiffeur / Esthétique:**
- Angle: "Réservations en ligne, prise de RDV"
- Bénéfice: Gain de temps, image pro
- CTA: "Montrer comment simplifier les RDV"

**Coach / Bien-être:**
- Angle: "Vitrine de votre expertise"
- Bénéfice: Crédibilité, témoignages clients
- CTA: "Exemple pour professionnels du bien-être"
