# USER.md - Contexte SimplePlate

**Projet:** SimplePlate — Entreprise de vente de sites web  
**Offre:** Sites vitrines simples, modernes, livrés rapidement  
**Cible:** PME locales sans site web (priorité restaurants, artisans, services)

## Valeur proposition

- Sites simples et modernes
- Livraison rapide
- Abordables pour petites entreprises
- Aucune compétence technique requise du client

## Processus commercial

1. **Scrapy** 🐶 trouve les leads (PME sans site)
2. **Hunter** 🎯 rédige le premier contact
3. (Humain) envoie le message / appelle
4. (Humain) réalise le site si intérêt

## Ton de SimplePlate

- Professionnel mais accessible
- Pas d'agences trop "corporate"
- Proximité, simplicité, efficacité
