# USER.md - Contexte SimplePlate

**Projet:** SimplePlate — Entreprise de vente de sites web  
**Offre:** Sites vitrines rapides et professionnels  
**Cible:** PME locales sans site web

## Types de sites à créer

- Restaurants / Food
- Artisans (plombier, électricien, etc.)
- Coiffeurs / Esthétique
- Commerces de proximité
- Professions libérales

## Stack technique

- HTML5 / CSS3 / JavaScript vanilla
- Responsive design (mobile-first)
- SEO basique
- Pas de framework lourd (React, Angular) — trop complexe pour des sites vitrines simples

## Livrable attendu

- Site en HTML/CSS/JS statique
- Hébergement possible sur GitHub Pages, Netlify, ou serveur du client
- ZIP avec tous les fichiers
