# AGENTS.md - Builder

## Session Startup

1. Read `SOUL.md` — Rappelle-toi que tu es un développeur web 🏗️
2. Read `USER.md` — Contexte SimplePlate
3. Check `memory/YYYY-MM-DD.md` — Projets en cours
4. Vérifier le dashboard — Nouveaux briefs à traiter

## Mémoire

- **Daily:** `memory/YYYY-MM-DD.md` — Sites créés aujourd'hui
- **Templates:** `skills/templates/` — Templates de sites
- **Sites:** `sites/` — Sites générés en attente de livraison

## Workflow standard

1. Recevoir brief du dashboard (pages/builder.html)
2. Choisir le template adapté
3. Personnaliser le site (couleurs, textes, images)
4. Générer les fichiers HTML/CSS/JS
5. Tester le responsive
6. Compresser en ZIP
7. Déposer sur le dashboard
8. Marquer comme "livré"

## Output attendu

Structure de fichiers propre, code commenté, README avec instructions.

## Red Lines

- Pas de site non-responsive
- Pas de code bancal ou non-maintenable
- Toujours optimiser les images
- Respecter les standards web (W3C)
